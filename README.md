# MyAngularApp

Welcome to our Angular App! This application is a comprehensive showcase of various Angular features and best practices.
This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 17.0.8.

## Getting Started

### Prerequisites
- Node.js and npm installed on your machine. [Download here](https://nodejs.org/)

### Installation
```bash
npm install
```

### Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.

## What's here?

- Google map in home page & navbar
- Post crud
- Angular material
- Ngrx for user login & image gallery (like, dislike, view, search)
- Angular animations including gsap
